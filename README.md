# 제주 야호~! 맛집 지도

구글시트에 있는 제주 맛집 리스트를 카드 형태 웹페이지로 보여주는 사이트입니다.

## 어떻게 자동으로 업데이트되나요?

`index.html` 안의 자바스크립트가 페이지를 열 때마다 구글시트를 CSV 형식으로 직접 읽어옵니다.
구글시트 → CSV 변환 주소는 이렇게 생겼습니다.

```
https://docs.google.com/spreadsheets/d/[시트ID]/gviz/tq?tqx=out:csv
```

그래서 구글시트에서 행을 추가하거나 수정하면, **웹페이지를 새로고침할 때 바로 반영**됩니다.
별도로 웹사이트를 다시 만들거나 배포할 필요가 없습니다.

이 방식이 동작하려면 시트 공유 설정이 "링크가 있는 모든 사용자 - 뷰어"로 되어 있어야 합니다.
(구글 계정 로그인 없이도 읽을 수 있는 정도의 공개 범위이며, 검색엔진에는 노출되지 않습니다.)

## 파일 구성

- `index.html` — 페이지 전체 (디자인 + 데이터 불러오기 로직이 한 파일에 모두 들어있습니다)

## 다른 시트로 바꾸고 싶다면

`index.html` 안에서 아래 부분의 `SHEET_ID` 값만 바꾸면 됩니다.

```js
const SHEET_ID = "1C4X2KjbE3sNUVSbSIjZjqo-oDFeEIhEEM8q0akxJsSg";
```

## 시트에 필요한 열(컬럼)

아래 다섯 개의 헤더 이름을 시트 어딘가에 그대로 두면, 웹페이지가 자동으로 그 위치를 찾아서 읽어옵니다
(행이나 열이 밀려 있어도 괜찮습니다).

- 구분 (카테고리: 베이커리, 양식, 일식, 중식, 카페, 한식, 키즈 등)
- 상호명
- 위치
- 지도 링크
- 비고

새로운 카테고리를 추가하면 자동으로 필터 버튼이 생기지만, 이모지/색상은 `index.html`의
`CATEGORY_META` 부분에 추가해야 예쁘게 표시됩니다. 등록되지 않은 카테고리는 기본 🍽️ 아이콘으로 표시됩니다.

## 문제 해결

페이지에 "맛집 리스트를 불러오지 못했어요"라는 메시지가 뜬다면:

1. 구글시트 공유 설정이 "링크가 있는 모든 사용자"로 되어 있는지 확인하세요.
2. 시트 안에 "구분", "상호명" 등의 헤더 이름이 정확히 들어있는지 확인하세요.
